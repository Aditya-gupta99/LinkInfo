# LinkInfo

Analyze your links effortlessly with LinkInfo! Get insights on top links, recent links, and usage. An Android application, written in Kotlin,seamlessly integrates with openinApp Api.

### Features
- Top Links: Identify your most popular links quickly.
- Recent Links: Keep track of the latest links you've shared.
- Usage Line Graphs: Visualize link usage trends over time with intuitive line graphs.
